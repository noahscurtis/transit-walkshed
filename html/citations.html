Data Sources & Citations

This page documents all datasets and tools used in the creation of our Transit Walkshed Analysis project.

Geospatial Datasets

  • Seattle City Limits (city_limits.geojson) Seattle GeoData Portal (City of Seattle)

    Used to clip transit stops and population layers to the Seattle boundary.

  • 2020 Census Population (2020population_4326.geojson, census.blocks.geojson) U.S. Census Bureau — TIGER/Line 2020 Census Blocks

    Processed in census.ipynb to extract population values and reproject to EPSG:4326.

    Supports population density visualization and statistics in walkshed outputs.

  • Link Light Rail Data (link_line.geojson, link_stops.geojson) Sound Transit GTFS Feed

    Stops derived from stops.txt; alignments from shapes.txt.

    Used to map current Link Light Rail segments and stations.

  • RapidRide Bus Data (rr_existing.geojson, rr_planned.geojson, RR_stops.geojson) King County Metro GTFS Feed

    Used to map existing and planned RapidRide routes and stops.

  • Transit Stops Within Seattle (RR_stops_citylimits.geojson)

    Derived internally from GTFS stop files and clipped using Seattle city boundaries.

    Provides stop-level accessibility input for walkshed calculations.

Tools, Libraries, and Web Services

  • Mapbox GL JS https://docs.mapbox.com/mapbox-gl-js/

    Interactive basemap, layers, and UI controls.

  • Turf.js https://turfjs.org/

    Used for geoprocessing operations such as buffers, unions, and intersections.

  • GitHub Pages

    Used to host the live site and serve GeoJSON files via HTTPS.
