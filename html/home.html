Seattle Transit Walkshed Analysis

An interactive spatial analysis exploring the population accessibility of Seattle's rapid transit network.

Project Overview

This project is part of the GEOG 328 Final and focuses on creating an interactive web map that visualizes the walkshed surrounding rapid transit stops in Seattle. By combining Turf.js spatial functions, Mapbox GL JS, and demographic datasets, the map identifies populations living within a 15-minute walking radius from each transit stop.

Core Functionality

The interactive map integrates multiple spatial layers—RapidRide routes, Link Light Rail stops, and U.S. Census population blocks—to calculate and display accessibility patterns across the city. A standard 0.75-mile buffer is used to approximate a 15-minute walkshed.

  • Hover interactions showing stop-specific statistics
  • Population density shading within walkshed boundaries
  • Census data joined to neighborhood basemaps
  • Isochrones and transit layers displayed in a cohesive visual layout

Data Sources

  • Sound Transit Open Transit Data
  • King County GIS Data
  • Seattle Open Data Portal
  • U.S. Census Bureau Population Data

Spatial Processing Workflow

  • Remove duplicate stops and standardize stop naming
  • Convert datasets into GeoJSON format
  • Reproject coordinates into WGS84 (EPSG:4326)
  • Clip census data to Seattle city limits
  • Apply spatial join with neighborhood layers
  • Generate 15-minute walkshed buffers
  • Intersect population blocks with walkshed regions

Referenced Tools & Documentation

  • Mapbox GL JS
  • Turf.js Spatial Analysis
  • Isochrone API
